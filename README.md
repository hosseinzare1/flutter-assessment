# Assessment

Flutter Assessment project.

## Getting Started

Flutter SDK version: 3.13.3

## Features
Clean architecture

### Packages
- Cache photos: Cached network image
- State management: Bloc
- Navigation: GoRouter
- Code Generation: Freezed
- Dependency injection: GetIt
